
## A simple example to implement Unit testing in laravel


### Steps to run

```
git clone https://github.com/gotocva/Laravel-Unit-Testing-example.git
cd Laravel-Unit-Testing-example
composer update
cp .env.example .env
php artisan migrate
php artisan test
```
